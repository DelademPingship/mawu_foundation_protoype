# Product Overview

The Mawu Foundation is a mission-driven organization focused on development work in Ghana's Volta Region with pan-African initiatives. This repository contains an investor demo showcasing the foundation's impact and programs.

## Core Purpose
- Highlight Ghana's Volta Region development work
- Showcase pan-African initiatives and impact
- Provide pathways for donations, volunteering, and partnerships
- Demonstrate transparency through financial reports and governance insights

## Key Features
- **Impact Storytelling**: Mission-driven hero experience with impact metrics
- **Program Showcases**: Education, Health, Water & Sanitation, Economic Empowerment, Community Development
- **Get Involved Hub**: Donation flows (Stripe-enabled), volunteer opportunities, partnerships
- **Transparency Center**: Financial reports, governance insights, trust-building content
- **Commerce Platform**: Merchandise showcase with upcoming payment integration
- **Content Management**: Blog updates, multimedia galleries, storytelling modules

## Current State
This is a static investor demo built with React + Vite + Tailwind. The backend infrastructure exists but the primary focus is on the static frontend experience for investor presentations and supporter engagement.

## Target Audience
- Potential investors and supporters
- Donors and volunteers
- Partners and collaborators
- Community members interested in the foundation's work
